This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

Visit this link:
[Vercel Link](https://vercel.com/fjasdojf-2974s-projects/english-lessons-maze-nextjs/deployments)
Basically you have just to push to master and it should re-deploy it. No need to run lint or build commands before.
Also no need to worry about dist or out folder. Or to edit further next.config.js
